для включения логирования закиньте в папку с игрой system.cfg и чтобы после выхода из игры и починки игры логирование чата не отключалось
в свойствах system.cfg поставьте галочку "Только для чтения" (пкм на system.cfg -> вкладка "Общие" -> поставить галочку "Только чтение")
это позволит сохранить логирование чата вкл.
